# Quickstart: Module 1 Content

## Prerequisites

- Node.js >= 18
- NPM

## Setup

1. Install dependencies:
   ```bash
   cd website
   npm install
   ```

2. Start development server:
   ```bash
   npm start
   ```

## Verification

1. Open `http://localhost:3000/docs/module-01/embodied-intelligence` (or similar path depending on sidebar).
2. Verify all interactive components render.
3. Check console for any PropType or React errors.
