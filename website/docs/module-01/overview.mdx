# Module 1: The Robotic Nervous System

Welcome to Module 1! In this module, we bridge the gap between the "Digital Brain" and the "Physical Body".

## Curriculum

1.  **[The Awakening](./embodied-intelligence)**: Understanding Physical AI and the Hardware Reality.
2.  **[ROS 2 Fundamentals](./ros2-fundamentals)**: The Nervous System of the robot.
3.  **[Anatomy of a Humanoid](./urdf-and-tf)**: Defining the robot body with URDF.
4.  **[Capstone: The Blind Walker](./capstone-project)**: Applying your skills to build a safety reflex.

## Goals

*   Understand the "Sense-Think-Act" loop.
*   Master the basics of ROS 2 nodes, topics, and services.
*   Visualize robot structures using URDF.
